# 💚 HairSafe - Your Hair Health Partner

## 🚀 Live Demo
[View Live Website](https://your-username.github.io/hairsafe)

## 📖 About
HairSafe is a comprehensive web application for hair wellness, offering:
- 🛒 E-commerce for hair care products
- 📞 Online consultation booking
- 🔐 User authentication with OTP
- 💳 Multiple payment options
- 📦 Order tracking
- 🌙 Dark mode support
- 👤 User profile with order history

## 🛠️ Technologies Used
- HTML5
- CSS3 (with Dark Mode)
- JavaScript (ES6+)
- LocalStorage for data persistence
- Swiper.js for carousel
- Font Awesome Icons
- Google Fonts
